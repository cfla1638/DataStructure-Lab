# 运算符优先级
|      | +    | -    | *    | /    | （   | ）   | ^    | #    |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| +    | >    | >    | <    | <    | <    | >    | <    | >    |
| -    | >    | >    | <    | <    | <    | >    | <    | >    |
| *    | >    | >    | >    | >    | <    | >    | <    | >    |
| /    | >    | >    | >    | >    | <    | >    | <    | >    |
| （   | <    | <    | <    | <    | <    | =    | <    | err  |
| ）   | >    | >    | >    | >    | err  | >    | >    | >    |
| ^    | >    | >    | >    | >    | <    | >    | >    | >    |
| #    | <    | <    | <    | <    | <    | err  | <    | =    |
